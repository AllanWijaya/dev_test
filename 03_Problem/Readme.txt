# Untuk menjalankan aplikasi, ketik perintah "npm start"


# Input dan Output Aplikasi

1. Input Pertama: 
Masukkan angka yang merepresentasikan panjang array yang akan diuji.  

2. Input Selanjutnya:  
Masukkan angka-angka sesuai jumlah panjang array yang telah ditentukan pada input pertama.  

3. Output:  
Program akan menampilkan angka yang hilang dari array tersebut.


# Contoh Penggunaan

Masukkan jumlah angka dalam array?
#Input pertama : 3

Masukkan angka ke-1?
#Input : 2

Masukkan angka ke-2?
#Input : 3

Masukkan angka ke-3?
#Input : 0

#Output : 1
