File `.env` digunakan untuk menyimpan konfigurasi lingkungan aplikasi.  

Variabel Lingkungan
1. PORT  
   - Deskripsi: Menentukan port yang digunakan oleh server aplikasi.  
   - Contoh: `PORT=3000`

2. DB_URI  
   - Deskripsi: URL koneksi ke database yang digunakan oleh aplikasi.  
   - Contoh: `DB_URI=mongodb://localhost:27017/mydatabase`



Untuk menjalankan aplikasi ketik "npm start" pada root directory
